# base\_partner\_surname\_opusvl

A port of `base_partner_surname` to OpenERP v7.

## Known limitations

The GUI for displaying them has room for a lot of improvement.
There are no labels on the First name and Last name fields
once they are filled in (the labels are currently placeholder text).
